# 🌦️ WeatherWise Bot — ваш умный помощник погоды

Краткое описание:  
**WeatherWise Bot** — это телеграм-бот для получения актуальной информации о погоде в любой точке мира. Проект создан для тех, кто ценит скорость, точность и красивую подачу данных. Бот поддерживает геолокацию, текстовый ввод города, графики температуры и уведомления о резких изменениях погоды.

> **Миссия проекта:** *«Сделать погоду не тем, что вы пережидаете, а тем, что вы планируете»* — основатель проекта А. Волков.

## О проекте

**WeatherWise Bot** решает простую задачу: даёт **точный** прогноз там, где вы уже есть — в мессенджере. Никаких лишних приложений, рекламы или сложных интерфейсов.  

Бот использует открытое API [OpenWeatherMap](https://openweathermap.org/) и кеширует запросы для снижения задержек. В основе лежит асинхронный код на `aiogram 3.x`.  

Проект поддерживает как *русский*, так и *английский* языки интерфейса.

## ✨ Возможности

- 🗺️ **Погода по геолокации** — отправьте геопозицию, бот определит город сам.  
- 🌍 **Поиск по названию** — поддерживаются города на русском и английском.  
- 📈 **График температуры на 5 дней** — встроенный `matplotlib` генерирует картинку.  
- ⏲️ **Уведомления о резкой смене погоды** (например, «через 2 часа дождь»).  
- 🌐 **Автоопределение языка** — если город введён кириллицей, ответ будет на русском.  
- 🧠 **Кеш запросов** — снижение нагрузки на API.  

**Вложенный список (структура команд бота):**
- Основные команды
  - `/start` — приветствие и краткая инструкция
  - `/help` — полный список команд
  - `/weather <город>` — текущая погода
- Расширенные команды
  - `/forecast <город>` — прогноз на 5 дней
  - `/alerts` — включить/выключить уведомления
  - `/lang` — смена языка
